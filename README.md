## Usage

### Install Requirements

```
pip install kamu-jp-modern aiohttp tqdm zstandard
```

### Command

```
python nextdrop.py send <target_ip> <file_path> [--threads <num_threads>] [--port <port>]
```

```
python nextdrop.py receive <save_dir> [--compress] [--port <port>]
```
